面向 Agent 执行的确定性验证。

相同的已签名授权 + 相同的链上结果 = 每一次都得到相同的判定。 不依赖模型。无需信任。证明上链到 ERC-8004。

确定性

A-G 检查是已签名 spec 与链上数据的纯函数。无随机性，不依赖模型。

可复现

任何人都可以重新运行同一项检查并得到相同的判定。信任来自可复现性。

独立

EIV 既不是钱包，也不是 Agent。它检查执行结果，而非推理过程。
